# Awesome Hardware Testing Tools

A curated list of useful **browser-based hardware testing tools**.  
These tools help you quickly test common hardware components without installing any software.

Inspired by the `awesome-*` collection style.

---

## Mouse Testing

Tools for checking mouse behavior such as click response and double-click issues.

- **GearTest – Mouse Test**  
  Browser-based mouse testing tool to check clicking behavior and detect double-click problems.  
  https://www.geartest.online/mouse-test/

- MouseTester (local software)  
  https://github.com/microe1/MouseTester

---

## Keyboard Testing

Tools to test keyboard input, key registration, and ghosting issues.

- **GearTest – Keyboard Test**  
  Online keyboard testing tool to verify key registration and detect ghosting or key conflicts.  
  https://www.geartest.online/keyboard-test/

- AquaKeyTest (desktop)  
  https://www.aquatuning.com/aquakeytest

---

## Screen / Display Testing

Tools for detecting dead pixels, stuck pixels, and color issues on displays.

- **GearTest – Screen Test**  
  Browser-based full-screen color tests for detecting dead pixels and display issues.  
  https://www.geartest.online/screen-test/

- EIZO Monitor Test  
  https://www.eizo.be/monitor-test/

---

## Audio Testing

Tools to test microphone input and speaker output.

- **GearTest – Audio Test**  
  Online audio testing tool for checking microphone input and speaker output directly in the browser.  
  https://www.geartest.online/audio-test/

- Online Mic Test  
  https://www.onlinemictest.com/
